# zenius-x-skul-id-rag-poc

Proof of concept for creating rag for chat bot zenius x skul id using clojure, open ai, and mongo db.

## Indexing in Mongo DB

```
db.runCommand({
    createSearchIndexes: "container-vectors",
    indexes: [
        {
            name: "container_vector_index",
            type: "vectorSearch",
            definition: {
                fields: [
                    {
                        type: "vector",
                        path: "embedding",
                        numDimensions: 1536,
                        similarity: "cosine"
                        }
                    ]
                }
            }
        ]
    });
```

## Usage

Run the project directly, via `:main-opts` (`-m indrabrata.zenius-x-skul-id-rag-poc`):

    $ clojure -M:run-m

Run that uberjar:

    $ java -jar target/zenius-x-skul-id-rag-poc-0.1.0-SNAPSHOT.jar
